# Digital E-Book Library (MERN Stack)

This project is a full-stack digital e-book library platform that allows users to explore, access, and manage books efficiently.

## 🚀 Live Demo
https://novelnest-ebookslibrary.netlify.app/

## 🛠️ Technologies Used
- React.js
- Node.js
- Express.js
- MongoDB

## ✨ Features
- User-friendly interface for browsing e-books
- Book management and upload functionality
- RESTful API-based backend architecture
- Responsive and clean UI design

## ⚠️ Note
The project is currently deployed and accessible via the live demo link.
